# 1.在图像分类任务中，以下哪种数据增强操作最能保证样本标签的语义一致性？
A随机水平翻转
B颜色通道反转（如RGB转BGR并反色）
C随机调整亮度至原亮度的5倍
D随机裁剪掉图像的70%区域
答案：A。解析：在图像分类任务中，语义一致性要求数据增强操作后图像的类别标签不变。A选项（随机水平翻转）通常不改变物体结构和类别，如翻转后猫的图片仍是猫，因此最能保证语义一致性。B选项（颜色通道反转如RGB转BGR并反色）会剧烈改变颜色外观，可能使图像难以识别（如红花变蓝花或负片效果），尤其对颜色敏感的类别影响大，无法保证一致性。C选项（随机调整亮度至原亮度的5倍）会导致过曝，细节丢失（如物体变为全白），破坏语义。D选项（随机裁剪掉图像的70%区域）可能移除关键部分（如只保留背景），只剩30%区域，无法识别物体，语义不一致。

# 2.在CNN网络中，图A经过核为3x3，步长为2的卷积层，ReLU激活函数层，BN层，以及一个步长为2，核为2×2的池化层后，再经过一个3*3的的卷积层，步长为1，此时的感受野是？
解析：记 r 为感受野，s 为步幅在原图上的“间隔”，k为卷积核。

- Conv3×3, 初始 R0=1，s0=1，stride1=2，k1=3: R1 = R0 + (k1−1)·s0 = 1+ （3-1）·1 = 3； s1= s0·stride1 = 2
- ReLU/BN：不变
- Pool2×2,          R1=3，s1=2，stride2=2，k2=2: R2 = R1 + (k2−1)·s1 = 3+ （2-1）·2= 5；  s2= s1·stride2 = 4
- Conv3×3,         R2=5，s2=4，stride3=1，k3=3: R3 = R2 + (k3−1)·s2 = 5+ （3-1）·4= 13；s3= s2·stride3 = 4
